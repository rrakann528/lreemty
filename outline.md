# موقع WatchTogether - هيكل المشروع

## نظرة عامة
منصة ويب لمشاهدة الفيديوهات مع الأصدقاء بشكل متزامن مع دردشة فورية، مشابهة لتطبيق Rave.

## هيكل الموقع

### 1. الصفحة الرئيسية (index.html)
- شاشة ترحيبية عصرية مع شعار الموقع
- زر "إنشاء غرفة جديدة"
- زر "الانضمام لغرفة موجودة"
- معاينة للمميزات الرئيسية

### 2. صفحة الغرفة (room.html)
- منطقة مشغل الفيديو الرئيسية
- مشغل فيديو مخصص مع أزرار تحكم (تشغيل/إيقاف، تقديم/رجوع، مستوى الصوت، ملء الشاشة)
- شريط إدخال رابط الفيديو
- قائمة مصادر الفيديو (YouTube، MP4، M3U8)
- منطقة الدردشة القابلة للطي
- قائمة المشاركين في الغرفة

### 3. صفحة حول الموقع (about.html)
- معلومات عن الموقع ومميزاته
- كيفية الاستخدام
- الأسئلة الشائعة

### 4. صفحة اتصل بنا (contact.html)
- نموذج تواصل
- معلومات التواصل

## المكونات الرئيسية

### 1. مشغل الفيديو
- مشغل فيديو HTML5 مخصص
- دعم YouTube (من خلال iframe API)
- دعم ملفات MP4 و M3U8
- أزرار تحكم كاملة
- وضع ملء الشاشة مع الحفاظ على الدردشة

### 2. نظام الدردشة
- دردشة فورية WebSocket
- إشعارات الرسائل الجديدة
- دعم الإيموجي
- عرض الصور والـ GIFs
- واجهة overlay في وضع ملء الشاشة

### 3. تزامن الفيديو
- مكتبة Socket.io للاتصال الفوري
- مزامنة حالة التشغيل
- مزامنة الوقت الحالي
- التعامل مع التأخير

### 4. إدارة الغرف
- إنشاء غرف فريدة
- دعوة الأصدقاء عبر الرابط
- قائمة المشاركين
- إعدادات الغرفة

## التقنيات المستخدمة

### Frontend
- HTML5 مع دعم PWA
- CSS3 مع Tailwind
- JavaScript ES6+
- Socket.io للاتصال الفوري
- WebRTC (اختياري)

### Backend
- Node.js مع Express
- Socket.io للاتصال الثنائي
- إدارة الغرف والمستخدمين

## المميزات المتقدمة

### 1. توافق الموبايل
- تصميم responsive كامل

### 2. دعم Apple/Safari
- استخدام تقنيات متوافقة
